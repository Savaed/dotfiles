# Dotfiles

A collection of dotfiles.

## Prerequisites

Make sure you have `git` and `stow` installed locally on your machine.

### Git

```
$ sudo pacman -S git
```

### Stow

```
$ sudo pacman -S stow
```

## Installation

Go to the `$HOME` directory. Then execute the following commands:

```
$ git clone https://github/Savaed/dotfiles
$ cd dotfiles
$ stow .
```
